## üöÄ Quick Usage

### **To View Individual Resume:**
```bash
# JSON Resume format
cat jsonresume_01_Rajesh_Kumar_Sharma.json | jq '.'

# Plain text format
cat resume_01_Rajesh_Kumar_Sharma.txt
```

### **To Preview as HTML:**
```bash
npm install -g resume-cli
resume serve jsonresume_01_Rajesh_Kumar_Sharma.json
```

### **To Export as PDF:**
```bash
resume export jsonresume_01_Rajesh_Kumar_Sharma.json --theme elegant
```
